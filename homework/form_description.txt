
Завдання: Створення форми реєстрації на подію

Мета:
Створити HTML-форму для реєстрації учасників у вигаданій події, використовуючи елементи <fieldset> та <legend> для кращої організації форми.

Вимоги:

Структура форми:

Створіть форму з елементами <fieldset> та <legend> для різних секцій:

Особиста інформація
Вподобання
У <fieldset> "Особиста інформація" додайте наступні поля:

Повне ім'я
Адреса електронної пошти
Номер телефону
Вибір "Розмір футболки" (Малий, Середній, Великий)
У <fieldset> "Вподобання" додайте:

Прапорці для "Діетичних уподобань" (Вегетаріанська, Веганська, Без глютену, Нічого)
Валідація форми:

Переконайтеся, що всі поля обов'язкові, крім "Номер телефону".
Перевірте правильність формату електронної адреси в полі електронної пошти.
Надайте відповідні повідомлення про помилки або показники для неправильних вводів.
Обробка відправки:

Додайте кнопку для відправлення форми.
Після відправлення відобразіть повідомлення про підтвердження або перенаправте на сторінку з подякою.
Додаткові вказівки:

Використовуйте теги <form>, <input>, <select> та <textarea> за потреби для елементів форми.
Використовуйте атрибути HTML, такі як required, type, pattern і т. д., для валідації.
Забезпечте належне підписування елементів за допомогою тегів <label> для кожного поля вводу.
Організуйте пов'язані поля за допомогою елементів <fieldset> та надайте <legend> для кожного <fieldset>.
Переконайтеся, що форма доступна і зручна для користувача.
Бонус (опціонально):

Додайте додаткові елементи форми, наприклад, вибір дати для вибору дати події.
Оформіть форму за допомогою CSS для поліпшення її вигляду.
Використайте JavaScript для надання зворотного зв'язку в режимі реального часу або покращення функціональності форми.
Подання завдання:
Поділіться файлом HTML або посиланням на живу веб-сторінку (якщо вона розміщена в мережі), щоб показати завершену форму реєстрації.
