Завдання: Створення розкладу занять

Мета:
Створити таблицю розкладу занять за допомогою HTML для відображення умовного розкладу університетських курсів.

Вимоги:

Структура таблиці:

Створіть таблицю для відображення розкладу занять.
Включіть стовпці для днів тижня (від понеділка до п'ятниці) та рядки для різних часових інтервалів (наприклад, з 8:00 до 18:00).
Вміст таблиці:

Заповніть таблицю умовною інформацією про курси (назви курсів, викладачів та номери кімнат).
Додайте щонайменше 3 курси для кожного дня на різних часових інтервалах.
Оформлення:

Використайте базовий CSS для оформлення таблиці для кращої зрозумілості та естетики (опціонально, але рекомендовано).
Застосуйте різні кольори фону для рядків, стовпців або конкретних комірок, щоб виділити курси.
Додаткові вказівки:

Використовуйте теги <table>, <tr>, <th> та <td> для створення структури таблиці.
Використовуйте відповідні атрибути, такі як colspan та rowspan, якщо це необхідно для об'єднання комірок.
Переконайтеся, що таблиця логічно організована і легко зрозуміла.
Розгляньте використання семантичних елементів HTML для покращення доступності.
Бонус (опціонально):

Реалізуйте ефекти при наведенні чи інші взаємодії CSS для виділення конкретних комірок чи рядків.
Зробіть таблицю адаптивною для менших екранів за допомогою медіазапитів CSS.
Додайте додаткову інформацію, таку як описи курсів або посилання, в комірки таблиці.
Відправлення:
Поділіться файлами HTML та CSS або посиланням на живу веб-сторінку (якщо вона розміщена в мережі), щоб продемонструвати завершену таблицю розкладу занять.
